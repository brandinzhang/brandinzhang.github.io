基本语法

##运算

  • 注释
  • 运算符
  • 逻辑操作 不可以用||、&&等c风格操作符。
  • 三目运算符

语句

条件判断

  • if结束要加:,逻辑归属由缩进决定而非大括号决定
  • assert这个关键词我们称之为“断言”，当这个关键词后边的条件为 False 时，程序自动崩溃并抛出AssertionError的异常。

循环语句

  • while语句
  • 当while循环正常执行完的情况下，执行else输出，如果while循环中执行了跳出循环的语句，比如 break，将不执行else代码块的内容
  • for循环是迭代循环，在Python中相当于一个通用的序列迭代器，可以遍历任何有序序列，如str、list、tuple等，也可以遍历任何可迭代对象，如dict。
  • range 函数
  • pass语句，在不进行任何操作的地方占位。
  • 推导式

数据结构

列表

  • 列表是有序集合，没有固定大小，能够保存任意数量任意类型的 Python 对象，语法为 [元素1, 元素2, ..., 元素n]。
  • 创建列表
  • 添加元素
  • 删除元素
  • 获取元素，通过元素的索引值，从列表获取单个元素，注意，列表索引值是从0开始的。
  • 通过将索引指定为-1，可让Python返回最后一个列表元素，索引 -2 返回倒数第二个列表元素，以此类推。
  • 切片的通用写法是 start : stop : step 左闭右开
  • 常用操作符

元组

  • Python 的元组与列表类似，不同之处在于tuple被创建后就不能对其进行修改，类似字符串。
  • 元组使用小括号，列表使用方括号。
  • 元组与列表类似，也用整数来对它进行索引 (indexing) 和切片 (slicing)。 ## 字符串

字典

字典是无序的键:值（key:value）对集合，键必须是互不相同的（在同一个字典之内）

面向对象

函数

基本内容

  • 函数以def关键词开头，后接函数名和圆括号()。

  • 函数执行的代码以冒号起始，并且缩进。

  • return [表达式] 结束函数，选择性地返回一个值给调用方。不带表达式的return相当于返回None。

  • 表达式

  • 函数文档

函数参数

  • 默认参数，与c++一样，必须是右边连续个默认参数，否则会报错，
  • 可变参数,就是传入的参数个数是可变的，*args - 可变参数，可以是从零个到任意个，自动组装成元组。
  • **kw - 关键字参数，可以是从零个到任意个，自动组装成字典。 ### 函数返回值

变量作用域

  • 与c++不同，Python 中，程序的变量并不是在哪个位置都可以访问的，访问权限决定于这个变量是在哪里赋值的。当内部作用域想修改外部作用域的变量时，就要用到global和nonlocal关键字
  • 函数递归 ython 的默认递归深度限制相对较低（通常在几千的数量级），可以通过 sys 模块中的 getrecursionlimit() 和 setrecursionlimit() 函数来查看和修改这个限制。

对象=属性＋方法

  • 继承

  • 类有一个名为__init__(self[, param1, param2...])的魔法方法，该方法在类实例化时会自动调用。__del__在对象被删除时自动调用。

  • 在 Python 中定义私有变量只需要在变量名或函数名前加上“__”两个下划线，那么这个函数或变量就会为私有的了。
